## Использование

Для запуска сидеров:
```bash
# Запуск всех сидеров
npm run db:seed

# Запуск конкретного сидера
npx sequelize-cli db:seed --seed 20251016153032-demo-simplified-data.js
```

Для отката сидеров:
```bash
npx sequelize-cli db:seed:undo:all
```
